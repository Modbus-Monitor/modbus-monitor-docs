### Activation Steps (Shop License)
1. Install XPF.
2. Open License / Activation dialog.
3. Paste license key from purchase email.
4. Click Activate (online). For offline: generate request → send to support → apply response file.
5. Restart app if prompted.

**Transfer to new PC:** Deactivate on old PC, then activate with same key on new PC. Contact support if hardware lock prevents activation.
