# ABB Modbus Maps

Modbus Monitor XPF includes pre-built ABB device maps to speed up commissioning, monitoring, and troubleshooting.

## Supported ABB Device Families

- M4M
- B23 / B24
- A41 / A42
- Trio 50 / Trio 60
