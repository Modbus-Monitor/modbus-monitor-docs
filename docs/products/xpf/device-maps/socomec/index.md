# Socomec Modbus Maps

Modbus Monitor XPF includes pre-built Socomec device maps to speed up commissioning, monitoring, and troubleshooting.

## Supported Socomec Device Families

- DIRIS A40 / A41 RS485
- DIRIS I31
- DIRIS U30
- Countis ECI3
- Countis ECI2
- DIRIS,D30,D50,D70
- DIRIS,D30,D50,D70 _
- Countis E63
- Countis E33, Countis E43
- NETYS PR,NETYS RT,ITYS
- Multis L20, Multis L40
- Countis E23
- Countis E24
- Countis E50 E53 E50R E53R
- DIRIS A20 modbus
- Countis E13
- Countis E14
- Masterys
- Countis E03
- Countis E04
- DIRIS U20
- DIRIS U10
