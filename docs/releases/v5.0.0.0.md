# Modbus Monitor XPF v5.0.0.0

**Release Date:** February 9, 2026  
**Compatibility:** Windows 10/11 (x86, x64, ARM64)

## 🚀 New Features
- Added complete HMI Dashboard system with 12 widget types
- Added enterprise licensing with Free and Professional tiers
- Added 300+ pre-built device maps (Schneider, ABB, Siemens, SolarEdge)
- Added state-based coloring and a range editor
- Added Trends widget with live charting and trend playback

## 🔧 Improved
- Improved Microsoft Store integration and purchase flow
- Added dark mode and theme support
- Made data grid operations thread-safe
- Built multi-platform support (x86, x64, ARM64)

## 🐛 Bug Fixes
- Fixed store initialization errors
- Fixed COM port configuration issues
- Fixed widget drag-and-drop behaviors
- Fixed platform architecture conflicts
