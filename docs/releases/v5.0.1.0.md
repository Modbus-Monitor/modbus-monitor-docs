# Modbus Monitor XPF v5.0.1.0

**Release Date:** February 20, 2026  
**Compatibility:** Windows 10/11 (x86, x64, ARM64)

## 🚀 Highlights
- HMI/Dashboard updates with layout and styling refresh
- Widget selection and gain behavior improvements
- Bar graph enhancements: bipolar support, center behavior, and clamping behavior
- Added row insert/add/paste-before-selected behavior
- Added progress bar on file load
- Internal security and performance enhancements

## 🔧 Improved
- Improved dashboard visual consistency and interaction flow
- Improved register table editing workflow for row operations

## 🐛 Fixed
- Fixed register scaling and precision behavior
- Decimal places no longer override register gain/offset precision and display
